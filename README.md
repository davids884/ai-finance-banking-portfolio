# AI Portfolio for Finance and Banking

**Author:** David Sanz

This repository is a portfolio project oriented to **AI / ML Engineering roles in financial services**, with a particular focus on **fraud detection, anomaly detection, and sequential risk modeling**.

## Repository contents
- `AI_Finance_Banking_Portfolio_David_Sanz_EN.ipynb` → main notebook in English
- `requirements_AI_Finance_Banking_EN.txt` → minimal dependencies
- `Guide_GitHub_LinkedIn_Submission_EN.md` → exact upload guide and click-by-click instructions

## Cases included
1. **Supervised fraud detection with an MLP**
   - Imbalanced binary classification
   - Threshold optimization on validation data
   - ROC-AUC, PR-AUC, Precision, Recall, and F1 evaluation

2. **Anomaly detection with an Autoencoder**
   - Reconstruction-based anomaly scoring
   - Training on normal observations only
   - Complementary signal for fraud monitoring

3. **Time-series risk modeling with a GRU**
   - Synthetic market regime simulation
   - Sequential feature engineering
   - Prediction of next-step high-volatility periods

## What this project demonstrates
- Deep learning on tabular financial-style data
- Unsupervised anomaly detection
- Sequential forecasting with recurrent neural networks
- Appropriate metric selection for imbalanced problems
- Business-oriented documentation for financial applications

## Positioning
This project was built to show the ability to translate AI methods into use cases relevant for:
